# GOLDEN HOUR
GOLDEN HOUR is a smart ambulance locator and tracking system that helps patients get emergency medical help within the golden hour. It allows users to find the nearest ambulance, request it quickly, and track it live until it reaches the hospital.

## Features
One-tap ambulance booking with GPS location.

Smart dispatch of the nearest ambulance using maps and traffic.

Live tracking, driver details, and ETA updates.

Hospital dashboard to see incoming patients and prepare in advance.

## Goal

Reduce ambulance delays and save lives by improving coordination between patients, ambulances, and hospitals during the golden hour.
